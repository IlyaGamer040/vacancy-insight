# Установка расширения в Яндекс.Браузер

1. Откройте Яндекс.Браузер.
2. Перейдите на страницу: `browser://extensions`.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное**.
5. Выберите папку `browser_extension` из этого проекта.

## Как пользоваться
- Откройте расширение и задайте критерии.
- Нажмите **Сохранить**.
- Нажмите **Проверить сейчас** или ждите автоматической проверки по интервалу.

## Примечания
- Бэкенд должен быть запущен и доступен по `http://localhost:8090`.
- Если порт другой, укажите новый в поле **API base URL**.
